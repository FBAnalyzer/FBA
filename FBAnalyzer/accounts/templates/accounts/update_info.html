New update info 26.9.2023


Download CSV feature added to Saved Games
  • You can now download the shot csv - file from Saved Games -page. Simply press the Download Shots - button after loading the game.


New update info 16.9.2023


NEW functionality added to Floorball Scanner.

Based on user requests we have updated the software to include ball possession gains and losses that can now be tagged to shotmap both line and player based.

New update functionalities:

Possession + and Possession -
  • Pressing the shotmap opens up a more extensive list of shot types, now with Possession + and Possession -. If Shooter tagging is on, it is also possible to tag which player lost or gained ball possession. Possession + and - only works for Team 1 and Ball possession selection does not affect the player selection but only Team 1 players can be selected.
  • Blue o symbol means gained ball possession (your player takes ball possession from the opponent), red o symbol means lost ball possession.
  • Player and line possession + and - are also added to Line and Player charts
Automatic Team 2 line change removed
  • Automatic Team 2 line change for 5-5 lines (lines 1 - 3) are removed, because it enables future development to calculate automatically number of shifts started with or without ball for lines.
  • Automatic Team 2 line change is still working for Powerplay and Shorthanded lines
Shooter and Passer tagging on Powerplay and 6v5
  • It is now possible to tag shooter and passer on Powerplay and 6v5 special teams.
  • Shot xG and pass xG are calculated on respective columns on line and player tables so they can be seen separately from 5-5 xG.
Improved charts and print view
  • Game and period charts are improved and print pdf file is also more classified and more clear.
Live page and Shooter tagging off by default
  • Live page and shooter and passer options are now off by default. If you want to send results to https://fbscanner.io/live - live page, select LIVE box on.
  • If you want to tag shooter, passer, and possession + and - by players, select Shooter&Passer box on.
Player line placement logic change
  • It is now possible to include a player in multiple lines. Player positions are no longer saved and reloaded, but you need to rebuild roster before each game. This is for simplicity.
